# ⚡ Python Shutdown Tool

Herramienta en **Python** con interfaz gráfica moderna que permite **programar apagados automáticos en Windows** de forma fácil y flexible.

La aplicación utiliza **CustomTkinter** para una interfaz moderna y **Windows Task Scheduler** para ejecutar los apagados programados.

---

# 🖥️ Interfaz

La aplicación incluye:

- Panel de configuración visual
- Tabla de tareas programadas
- Edición de tareas existentes
- Eliminación de tareas
- Consulta detallada de tareas
- Aviso automático al iniciar sesión

---

# 🚀 Funciones

### ⏰ Programación de apagado

Permite programar apagado de varias formas:

- 📅 **Fecha y hora exacta**
- ⌛ **En X horas**
- 🔁 **Diario**
- 🔄 **Cada N horas**

---

### 🧠 Administración de tareas

- Crear tareas
- Editar tareas existentes
- Eliminar tareas
- Ver detalles completos
- Actualizar lista

---

### 🔔 Aviso al iniciar sesión

La herramienta puede mostrar un **mensaje automático al iniciar Windows** para recordar que el equipo tiene apagados programados.

---

# 📦 Tecnologías usadas

- **Python 3**
- **CustomTkinter**
- **Tkinter**
- **Windows Task Scheduler**
- **JSON storage**
